This is a simple program to solve quadratic functions using the Bhaskara method.

### How to run
1. Open the terminal
   
   - Windows
   ```
   Windows + R : "cmd" or "powershell"
   ```
   
   - Linux
   ```
   Ctrl + Alt + T
   ```
   
2. Insert the code and run
   ```
   node <script-path>
   ```
   
3. Insert the values in A, B and C and see the results...
