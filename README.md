
### Quadratic Function Calculator
This is a simple program to solve quadratic functions using the Bhaskara method.

---

### Dependences
This is dependences to run the program:

| Name       | Download Link                                  |
|------------|------------------------------------------------|
| NodeJS     | https://nodejs.org/en/download/package-manager |
| TypeScript | https://www.typescriptlang.org/download/       |

### Installing

#### NodeJS
   1. Open the terminal
   
      - Windows
      ```
      Windows + R : "cmd" or "powershell"
      ```
   
      - Linux
      ```
      Ctrl + Alt + T
      ```
   
   2. Insert the following command

      ```
      cd <program-file-path>
      ```

   3. Then
      ```
      npm init -y
      ```

#### TypeScript
   1. Open the terminal
   
      - Windows
      ```
      Windows + R : "cmd" or "powershell"
      ```
   
      - Linux
      ```
      Ctrl + Alt + T
      ```
   
   2. Insert the following command

      ```
      cd <program-file-path>
      ```

   3. Then
      ```
      npm install -g typescript
      ```


---

### How to run
1. Open the terminal
   
   - Windows
   ```
   Windows + R : "cmd" or "powershell"
   ```
   
   - Linux
   ```
   Ctrl + Alt + T
   ```
   
2. Insert the following command and run
   ```
   node <script-path>
   ```
   
3. Insert the values in A, B and C and see the results...
   ```
   ===================
    QUADRATIC FUNCTION
       CALCULATOR
   ===================
   Enter a value for A: <...>
   Enter a value for B: <...>
   Enter a value for C: <...>
   ```
